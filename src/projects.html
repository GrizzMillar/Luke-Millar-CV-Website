Uniframe

UniFrame is an easy to use tool specifically designed for python modular unit test framework’s with remote testing capabilities via SSH and enhanced reporting and debugging capabilities. A tool that sits closer to the implemented framework will offer a more streamlined user-friendly experience by focusing solely on the needs and workflows of the test framework as opposed to a tool like Jenkins that offers a broad scope of complexity for building and deploying. This tool will automate the software testing process while minimising some of the limitations of industry tools that I have identified. The tool will have standard configuration templates making it easy to use without extensive knowledge for configuring it.
