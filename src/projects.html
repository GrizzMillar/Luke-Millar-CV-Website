Uniframe

UniFrame is an easy to use SSH tool for the python unit test framework with enhanced reporting and debugging capabilities. UniFrame offers a more streamlined user-friendly experience executing unit tests via SSH by focusing solely on the test framework as opposed to using an automation tool like Jenkins that requires the setup of multiple SSH plugins, credentials and Nodes. The tool has standard configuration templates making it easy to use without extensive knowledge for configuring the SSH connection. Some templates will suit smaller and less complex projects while others will be more advanced for extra configuration. UniFrame automates the setup process of test environments installing any dependencies and setting any environment variables. This tool is available in the form of a CLI tool and as a web application.

Dialed-In

Dialed-In is a performance analysis application for golfers. It allows golfers to track their scores and statistics in a variety of ways. Based on the PGA Tour’s statistic formula Dialed In uses GPS so golfers can track shot by shot in real time during their round. Representation of statistics can either be predefined or customized by users to a variety of specific metrics and timelines. Furthermore this application has a social media aspect were golfers can befriend and connect with others and view each others statistical track record. The app is available as an IOS application.
